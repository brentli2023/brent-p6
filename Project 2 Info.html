This is Project #2 of my portfolio! For this project, I created a game using Scratch! The purpose of Scratch is to show how coding fundamentally looks like to beginners and children. It is a visual interface that allows creativity to surface to create stories and games like mine. To start playing, load the code by clicking the green flag in the top right corner of the screen. Then, just follow the directions!

Here is a link to download the scratch file: My Scratch!
Go to the Scratch website and open the downloaded file and then play!
